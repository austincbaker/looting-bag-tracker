# Looting Bag Loot Tracker

Tracks loot that goes into an open looting bag during chest and container interactions.

The standard RuneLite Loot Tracker does not record items that go directly into a looting bag instead of the player's inventory. This plugin fills that gap by detecting when you open a chest or container while holding an open looting bag, then recording any new items that appear in the bag under the source's name in the Loot Tracker panel.

## Supported interactions

- Rogues' Den chests (Search for traps)
- Standard chests (Open, Unlock, Search)
- Pickpocketing (Steal)
- Other containers (Loot)

## Requirements

- Looting Bag must be open (not closed) in your inventory
- Loot Tracker plugin must be enabled
